== Installation ==

1. Upload the folder `jbreadcrumb-aink` to your `/wp-content/plugins/` directory,
2. Activate the plugin through the "Plugins" menu in WordPress,
3. Configure options via the jBreadCrumb ==> Settings menu,

	&lt;?php if (function_exists('jBreadCrumbAink')) { echo jBreadCrumbAink(); } ?&gt;
